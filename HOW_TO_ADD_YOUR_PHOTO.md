# How to Add Your Profile Photo

## 🚨 IMPORTANT: Your photo is NOT automatically added - you need to do this manually!

## Step-by-Step Instructions:

### 1. Save Your Photo
- Right-click on your profile photo (the one you shared)
- Save it to your computer
- Rename it to: **`mohamed-ashif.jpg`** (exactly this name)

### 2. Add to Project
Navigate to your project folder and place the photo here:
```
C:\Users\madashhh\Documents\Kiro\resume\public\images\profile\mohamed-ashif.jpg
```

### 3. Verify the File
Make sure the file is:
- Named exactly: `mohamed-ashif.jpg`
- Located in: `public/images/profile/`
- Format: JPG or PNG
- Size: At least 400x400px

### 4. Restart Dev Server
```bash
# Stop the current server (Ctrl+C)
# Then restart:
pnpm dev
```

### 5. Refresh Browser
- Open http://localhost:3000
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

## Need Help?

If the photo still doesn't show after following these steps:
1. Check the file name is exactly: `mohamed-ashif.jpg`
2. Check it's in the correct folder: `public/images/profile/`
3. Try renaming it to `mohamed-ashif.png` if it's a PNG file
4. Clear your browser cache
